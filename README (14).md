# Age Calculator

A simple web-based Age Calculator developed using HTML, CSS, and JavaScript for CodeAlpha Internship.

## Features

- User enters Date of Birth
- Calculates exact age in:
  - Years
  - Months
  - Days
- Clean and responsive UI
- Beginner-friendly project

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Used

- DOM Manipulation
- JavaScript Date Object
- Input Validation

## Project Structure

```plaintext
CodeAlpha_AgeCalculator
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`

OR

Run using VS Code Live Server.

## Output

The calculator displays the user's age in Years, Months, and Days.
